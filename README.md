# ascii-map

A terminal-based ASCII map explorer.

## Install

```bash
brew install luthiraa/ascii-map/ascii-map
```

## Run

```bash
ascii-map
```

## Controls

| Key | Action |
| :--- | :--- |
| **WASD** / **Arrows** | Pan map |
| **+** / **-** | Zoom in / out |
| **/** | Search location |
| **n** | Toggle street names |
| **[** / **]** | Adjust aspect ratio |
| **r** | Reset view |
| **q** | Quit |
